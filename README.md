# SOMAS Maps: Survey Location Mapping Toolkit

This repository contains a dual-purpose toolkit designed to generate high-quality, publication-ready geographic maps for survey locations along the US East Coast. 

It provides both **Python-based automated rendering scripts** for highly customized, static spatial layouts, and a **Zero-Installation Interactive Web Viewer** that allows collaborators to create and export custom maps entirely within their web browser.

## 🛠️ Technologies Used

**Python Environment (v3.12)**
* **Matplotlib:** For core plotting and figure generation.
* **Cartopy:** For accurate geospatial data processing and geographic projections.
* **NumPy:** For coordinate and scale bar mathematical calculations.

**Interactive Web Environment**
* **HTML5 / CSS3 / JavaScript:** Core web technologies (No server required).
* **Leaflet.js:** A lightweight, powerful interactive mapping library.
* **TopoJSON Client:** To dynamically render high-resolution physical landmasses and political borders.
* **html2canvas:** To capture and export the browser's DOM elements directly into downloadable PNG images.

## 🗺️ How to Use the Interactive Map Viewer (`SurveyMapViewer.html`)

The crown jewel of this toolkit for non-coders is `SurveyMapViewer.html`. It replicates the strict grayscale, publication-ready aesthetic of the Cartopy Python scripts but requires **absolutely no installation, coding, or Python environment**. 

**To use:** Simply double-click `SurveyMapViewer.html` on any Windows, Mac, or Linux machine. It will open securely in your default web browser.

### Features & Usage:
* **Add Survey Points:** You can add points by typing exact Latitude/Longitude coordinates into the sidebar, or simply **clicking anywhere directly on the map**. 
* **Remove Points:** Click on any existing dot on the map and press the red `🗑️ Remove Point` button inside its popup.
* **Dynamic Cartography:** Use the sidebar checkboxes to toggle map features on the fly. You can switch between raw landmasses, show/hide state and country borders, enable background city names, or overlay a dynamic latitude/longitude grid that automatically adjusts its spacing based on your zoom level.
* **Smooth Zooming:** The map is configured to allow fractional zoom increments (0.10 steps), giving you buttery-smooth control over the exact framing of your map.
* **💾 Save as Image:** Once your map is perfectly framed, click the blue **Save as Image** button. A script (`html2canvas`) will instantly take a high-resolution snapshot of your current map view—including your grid lines, scale bar, and North arrow—and download it straight to your computer as a `.png` file named with today's date.

## 🐍 How to Use the Python Scripts

For advanced programmatic control, use the Python scripts. Ensure you have Python 3.12 installed and set up your virtual environment.

**1. Install Dependencies**
Because Cartopy relies on underlying C-libraries (GEOS and PROJ), it is highly recommended to use `conda` for environment setup, or standard `pip` if your system wheels are configured:
```bash
pip install -r requirements.txt
```

**2. Generate Multi-Panel Maps**
To generate the primary map featuring Long Island and Virginia zoomed-insets connected via dynamic bounding boxes:
```bash
python create_maps.py
```

**3. Generate Batch Regional Maps**
To iterate through predefined extents (e.g., Peconic Bay, New York State, Custom Mid-Atlantic bounds) and generate a suite of standardized maps:
```bash
python create_add_maps.py
```
*Note: Maps generated by the batch script are automatically saved into the `map_outputs/` directory.*
